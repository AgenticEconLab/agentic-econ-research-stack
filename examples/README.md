# Economics Research Agent Examples

## Overview

Complete, working examples of agentic AI systems for economics research. These examples demonstrate real-world implementations combining frameworks, tools, and patterns.

**Note**: While these examples focus on economics research, they can be adapted for finance, political science, and other research disciplines.

## Examples

### [Economics Literature Review Agent](literature-review-agent.md)
Automated economics literature review system that searches, summarizes, and synthesizes economics research papers.

### [Econometric Data Analysis Agent](data-analysis-agent.md)
Agent that loads, analyzes, and visualizes economics data with econometric analysis.

### [Economics Hypothesis Testing Agent](hypothesis-testing-agent.md)
Agent that generates economics hypotheses, designs experiments, and tests them statistically.

## Using the Examples

Each example includes:
- Complete code implementation
- Framework-specific versions
- Step-by-step explanations
- Best practices and tips
- Common pitfalls and solutions

## Prerequisites

- Python 3.8+
- API keys for LLM providers (OpenAI, Anthropic, etc.)
- Required packages (see individual examples)
