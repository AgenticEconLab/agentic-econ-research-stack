# LLM Providers

## Overview

LLM (Large Language Model) providers offer APIs and services to access language models for building agentic AI applications for economics research. Choosing the right provider depends on your needs for cost, performance, privacy, and features.

## Provider Categories

### Commercial APIs
- **[OpenAI](openai.md)**: GPT-4, GPT-3.5, most popular
- **[Anthropic](anthropic.md)**: Claude, strong reasoning capabilities

### Open Source
- **[Open Source Models](open-source.md)**: Self-hosted models, privacy-focused

## Choosing a Provider for Economics Research

| Feature | OpenAI | Anthropic | Open Source |
|---------|--------|-----------|-------------|
| Ease of Use | Excellent | Excellent | Medium |
| Cost | Medium | Medium | Low (hosting) |
| Performance | Excellent | Excellent | Varies |
| Privacy | API | API | Full Control |
| Best For | General Economics Use | Economics Reasoning | Privacy |

## Common Use Cases in Economics Research

- **Economics Agent Reasoning**: Models that can reason through complex economics problems
- **Econometric Code Generation**: Models for generating econometric analysis code
- **Economics Text Analysis**: Models for analyzing economics research papers
- **Economics Conversation**: Models for interactive economics research assistants

**Note**: These use cases apply to finance, political science, and other research disciplines.
